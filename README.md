Kyra Hanson - Budget Expense Program

How to Run Program:
1. Open IDE like Visual Studion
2. Open BudgetCalculator.c in VS
3. Compile using gcc BudgetCalculator.c -o budget -lm
4. Run program using ./budget, and press enter to run

How Program Works:
User input: Prompts user to enter in their paychecks and monthly expenses
2. Double data type: Double data type/float data type is used for decimal points for dollar and cents
3. Calculation: Adds paycheck amounts together for total monthly income, finds the total amount spent per month, and calculates which percentage of income goes towards each expense, as well as any leftover money from monthly income
4. Final results: Once the percentages and leftover amount have been calculated, the results are displayed on screen
